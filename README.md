# TicTacToe

This project is a basic web-based Tic-Tac-Toe game. I copied this program as an exercise to better understand how GitHub Copilot works.

## Reference

The original source for this project is from [this Gist by olg200492](https://gist.github.com/olg200492/efb09eec592c9a500865eaa48a84d494), which itself credits [Web Dev Simplified's CodePen](https://codepen.io/WebDevSimplified/pen/MWYGogW).

## Purpose

The first test is to see if I can update the README in a separate branch and merge it back into the main branch using a pull request.

## Structure

- `public/index.html`: Main HTML file
- `src/style.css`: Stylesheet
- `src/script.js`: Game logic
- `public/assets/`: (Optional) Images, icons, etc.

## Usage

Open `public/index.html` in your browser to play the game.
